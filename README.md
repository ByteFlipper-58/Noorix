## Overview
Noorix is a modern, responsive web application designed to help Muslims track prayer times, Qibla direction, Ramadan schedule, and other essential information. Built with React, TypeScript, and Tailwind CSS, it offers a seamless user experience across all devices.

## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/byteflipper-58/noorix.git
   cd noorix
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create `.env` file with your Firebase config (see `.env.example`).

4. Run development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:5173](http://localhost:5173).
